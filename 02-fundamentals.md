## HTML
HTML(超文本标记语言, HyperText Markup Language)用于结构化网页内容。最简单的HTML页面如下

{% highlight html %}
<html>
<head>
	<title>Page Title</title>
</head>
<body>
	<h1>Page Title</h1>
	<p>This is a really interesting paragraph.</p>
</body>
</html> 
{% endhighlight %}
